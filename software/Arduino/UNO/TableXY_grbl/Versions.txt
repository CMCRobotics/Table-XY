Version: 0.3

Date:	22/03/2026
Author: Jose Luis Gomez Costa

	The previous approach is too complex, too many errors when modifying the actual processing 
	
	A  simpler one is chosen:
		When Z command is detected:
			- Mxx command is executed
			- then the original command line is executed unmodified.
		Process: 
			1.0 Keep OriginalLine unmodified
			2.0 Command Line is parsed:	
				2.1 if no Z command is found, jump to 3.0
				2.2 if Z command is found: Generate a Dummy CommandLine with the M03 or M05
				2.3 Execute Dummy CommandLine
				2.3.1 If error detected, jump in 4.0
			3.0 Execute OriginalLine
			4.0 Return with:
				detected error
				or OK if none
		
	This will imply always processing the Z axis, but as there is no hardware it will not interfere with the actual behaviour with a Pencil.

	OriginalLine: one line of 0-terminated G-Code.
		The line is assumed to contain only uppercase characters and signed floating point values (no white-space).
		Comments and block delete characters have been removed.
		
	Zvalue
		value = floating Point
		value >= 2.0 -> Pencil Up
		value  < 2.0 -> Pencil Down
		
		
	pencil.h:
		All definitions for the Pencil behaviour
