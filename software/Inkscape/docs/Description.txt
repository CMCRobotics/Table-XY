Multi shape_0029.ngc:
	G00 Z1.000000: Z1.000000 monter l’outil à +1mm: zone livre

	G01 Z-1.000000 F100.0(Penetrate): Z-1.000000 descendre l’outil à -1mm, Fxxx: vitesse de rotation de l’outil

	G01 X129.643569 Y135.608775 Z-1.000000 F400.000000: déplacer l’outil dans l'axe XY, l’outil reste a -1mm


gcodetools.py:

	options.Zsafe: 
		-> Zsafe: auteur à partir de laquelle il est possible de bouger l’outil sans tracer (dans l’exemple: 1.00 mm). Ceci c'est une option qui est choisie dans Inkscape menu: Extensions/GCodeTools/Chemin vers GCode/Preferences
	options.Zdepth 
		-> Zdepth: profondeur à laquelle descendre l’outil pour tracer (dans l’exemple: -1.00 mm). Ceci c'est une option qui est choisie dans Inkscape, et transféré dans le fichier Multi shape.svg.
